壳边-壳边绑定






壳边-壳边绑定（TieEdge）的定义（对应上图）：

①主功能工具栏点击“连接”图标；

②点击“新建”连接；

③连接类型选择“壳边-壳边绑定”；

④节点选取方式设置为“拾取”，Shift+鼠标左键点选PART1的边节点作为“从边节点集”，Shift+鼠标左键自上而下或从下到上按顺序点选与PART1相邻的PART2边节点作为“主边节点集”，定义”失效塑性应变值”，连接处节点塑性应变超过该阈值时，连接失效；

⑤点击“确定”按钮，完成壳边-壳边绑定的定义。

关键字说明：

当Type=TieEdge时:

第一行：1）从边节点集合名，2）主边节点集合名。

第二行：基于节点等效塑性应变失效的应变阈值（默认阈值为0，若为默认值，则不考虑连接失效）。

Mx文件示例：

*connector, type = tieedge

nset7,nset8

10

*set, type = node, name = nset7

1232, 1243, 1254, 1265, 1276, 1287, 1298, 1309

1320, 1331, 1221

*set, type = node, name = nset8

2575, 2564, 2553, 2542, 2586, 2597, 2608, 2619

2630, 2641, 2652

（蓝色部分为集合，详集合定义章节）
