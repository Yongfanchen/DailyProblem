刚性墙






刚性墙（rigidwall）的定义（对应上图）：

①主功能工具栏点击“刚性墙”图标，进入刚性墙创建栏；

②刚性墙类型选择“无限大平面刚性墙”；

③在法向起始坐标栏(x,y,z)输入”0,0,0”，在法向终止坐标栏(x,y,z)输入”0,12,0”，以此来定义刚性墙的法向；（注意坐标用英文逗号隔开）；

④点击“确定”完成无限大平面刚性墙的定义。

关键字说明：

*rigidwall：定义刚性墙

1.必须定义的属性设置:

Type

当type=planar时，定义无限大的平面刚性墙。

当type=flat时，定义有限大小的平面的刚性墙。

当type=prism时，定义长方体的刚性墙。

当type=cylinder时，定义圆柱状的刚性墙。

2.可选的属性设置:

Fun

fun=(curve_name)，定义刚性墙运动曲线（type!=planar）。

Moving

当moving=velocity时，定义刚性墙的速度（type!=planar）；

当moving=displacement时，定义刚性墙的位移（type!=planar）。

3.参数行

当type=planar时：

第一行：

1.刚性墙法向的起始坐标（x, y, z），

2.刚性墙的法向终止坐标（x, y, z），

3.静摩擦系数，

4.刚性墙在法向方向移动的速度值。

当type=flat时：

第一行：

1.刚性墙法向的起始坐标（x, y, z）；

2.刚性墙的法向终止坐标（x, y, z）；

3.静摩擦系数。

第二行：

4.刚性墙边向量的终止坐标（x, y, z），起始坐标与法向起始坐标相同；

5.刚性墙的边长L（上一步定义的边向量所在的边）；

6.刚性墙的边长M。

第三行，当属性moving定义时才使用第三行：

7.刚性墙运动速度或位移的方向余弦（vx, vy, vz）。

当type=cylinder时：

第一行：

1.刚性墙法向的起始坐标（x, y, z）；

2.刚性墙的法向终止坐标（x, y, z）；

3.静摩擦系数。

第二行：

4.半径

5.轴向长度

第三行，当属性moving定义时才使用第三行：

6.刚性墙运动速度或位移的方向余弦（vx, vy, vz）。

当type=Prism时：

第一行：

8.刚性墙法向的起始坐标（x, y, z）；

9.刚性墙的法向终止坐标（x, y, z）；

静摩擦系数。

第二行：

11.刚性墙边向量的终止坐标（x, y, z），起始坐标与法向起始坐标相同；

12.长度L

13.长度M

14.长度P

第三行，当属性moving定义时才使用第三行：

刚性墙运动速度或位移的方向余弦（vx, vy, vz）

Mx文件示例：

*rigidwall, type = planar

0,0,0, 0,12,0,0,0
