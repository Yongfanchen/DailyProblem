齿轮-齿条传动






齿轮-齿条传动（RackPinionJoint）的定义（对应上图）：

①主功能工具栏点击“连接”图标；

②点击“新建”连接；

③连接类型选择“齿轮-齿条传动”；

④依次定义节点1~节点6，其中节点1、节点3及节点5从属于PART-A（齿轮），节点2、节点4及节点6从属于PART-B（齿条），节点1、3定义齿轮的法向，节点2、6定义齿条平面法向，节点2、4定义齿条的移动方向，节点1、5定义齿轮所在平面。这样方可实现PART-A和PART-B的齿轮-齿条传动连接定义；

⑤点击“确定”按钮，完成齿轮-齿条传动的定义。

关键字说明：

当Type=RackPinionJoint时：

第一行：1）节点1，2）节点2，3）节点3，4）节点4，5）节点5，6）节点6，7）阻尼因子（默认值1.0），8）罚因子调整项（默认值1.0）；

第二行：1）齿轮分度圆半径。

Mx文件示例：

*connector, type = RackPinionJoint

7300, 7301, 7304, 7303, 7302, 7305, 1.000000,1.000000

100.000000

*connector, type = extranode

part3, nset20

*connector, type = extranode

part4, nset21

*set,type = node, name = nset20

7300, 7304, 7302

*set,type = node, name = nset21

7303, 7305, 7301
