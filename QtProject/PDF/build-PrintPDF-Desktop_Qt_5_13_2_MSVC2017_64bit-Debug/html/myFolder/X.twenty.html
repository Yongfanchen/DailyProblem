输出


输出（output）的设置（对应上图）：

①主功能工具栏点击“输出”图标；

②点击“新建”名称为“FieldOutput_1”的输出名称；

③输出类型选择为“场变量”或者“历程变量”，并定义数据“输出时间”间隔；

④输出类型可选择“节点”或者“单元”；

⑤设置“输出场变量”，灰色部分为默认输出的场变量；

⑥点击“确定”，完成场变量的输出设置。

关键字说明：

*output：定义输出

1.必须定义的属性设置:

mode

当mode=mxdb，定义输出文件格式是mxdb场输出格式。

当mode=mxh，定义输出文件格式是mxh历程数据输出格式。

2.可选的属性

name

定义该输出的名称。

（以下参数frequency和time interval是互斥的参数，当两者都未定义时，默认输出20帧计算结果）

time interval

定义输出文件的输出时间间隔。

frequency

定义输出文件的输出频率。

3.参数行

无

*field：定义场变量输出

1.必须定义的属性设置:

object

当object=node，定义节点对象的场输出；

当object=element，定义单元对象的场输出。

2.可选的属性

set

定义输出对象集合，该属性缺省时，默认输出对象集合是整体模型。

variable

当variable=default，定义当前输出对象下对应的默认场变量的输出；

当variable=all，定义当前输出对象下对应的所有场变量的输出。

3.参数行

输出变量标识符（逗号隔开，或者换行隔开，每行最多可打印8个变量名称），当variable属性缺省时，参数行不可为空。


*history：定义历史变量输出

1.必须定义的属性设置:

object

当object=node，定义节点对象的历程输出；

当object=element，定义单元对象的历程输出。

当object=energy，定义能量的历程输出。

当object=mat，定义部件的能量、刚体位移的历程输出

当object=RigidWall，定义刚性墙的合力输出

当object=SectionForce，定义截面力的输出

set（仅当object=节点和单元时是必选项，否则为可选项）

定义输出对象集合。

2.可选的属性

variable

当variable=default，定义当前输出对象下对应的默认历程变量的输出；

当variable=all，定义当前输出对象下对应的所有历程变量的输出。

3.参数行

输出变量标识符（逗号隔开，每行最多可打印8个变量名称），当variable属性缺省时，参数行不可为空。

Mx文件示例：

*output, mode = mxdb, name = FieldOutput_1, timeInterval = 10

*field, object = element, variable = all
