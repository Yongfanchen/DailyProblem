定义分析类型


分析类型的定义步骤（对应上图）

①主功能工具栏点击“分析类型”图标；

②点击“新建”输入分析类型名称为“Case_1”；

③选择分析类型，本算例为“显式动态分析”；

④定义“终止时间”、“质量缩放控制设置”及“安全因子”；

⑤点击“确定”按钮，完成分析类型的定义。

关键字说明：

1.必须的属性设置：

Type

当type=explicitdynamic，定义显式动态分析不包含SPH模块分析）。

当type=SPHExplicitDynamics，定义SPH-FEM耦合动态分析。

当type=fluid，定义显式流体分析。

2.可选的属性设置：

name

用于定义分析类型的名称。

Mx文件示例：

*explicit_analysis, type = explicitdynamics, name = Case_1

*timestep

,1

*autostepcontrol

0, 100, 0.8
