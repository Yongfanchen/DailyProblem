转动副






转动副（RevolutJoint）的定义（对应上图）：

①主功能工具栏点击“连接”图标；

②点击“新建”连接；

③连接类型选择“转动副”；

④依次定义节点1~节点4，其中节点1和节点3从属于PART-A，节点2和节点4从属于PART-B，节点1和节点3定义转动副的旋转轴。在空间坐标上，节点1、2重合，节点3、4重合。这样方可实现PART-A和PART-B的转动副连接定义；

⑤点击“确定”按钮，完成转动副的定义。

关键字说明：

当Type=RevolutJoint时：

第一行：1）节点1，2）节点2，3）节点3，4）节点4，5）阻尼因子（默认值1.0），6）罚因子调整项（默认值1.0）；

Mx文件示例：

*connector, type = revolutejoint

5000,5001,5002,5003,1,1

*connector, type = extranode

part3,nset7

*set, type = node, name = nset7

5000,5002

*connector, type = extranode

part4,nset8

*set, type = node, name = nset8

5001,5003

（蓝色部分为集合，详集合定义章节）
