点焊






点焊（spotweld）的定义（对应上图）：

①主功能工具栏点击“连接”图标；

②点击“新建”连接；

③连接类型选择“点焊”；

④鼠标点击“主节点”，Shift+鼠标左键点选模型节点（1331）作为主节点；同理，鼠标点击“从节点”，Shift+鼠标左键点选模型节点（2531）作为从节点；

⑤定义点焊的失效参数；

⑥点击“确定”按钮，完成点焊连接的定义。

关键字说明：

当Type=spotweld时

第一行：1）从节点，2）主节点，3）失效法向力，4）失效切向力，5）法向失效因子，6）切向失效因子，7）失效时间，8）失效有效塑性应变

Mx文件示例：

*connector, type = spotweld

1331,2652,0,0,0,0,1e+20,1e+20

齿轮传动（GearJoint）的定义（对应上图）：

平动约束






平动约束（translation）的定义（对应上图）：

①主功能工具栏点击“连接”图标；

③连接类型选择“平动约束”；

④Shift+鼠标左键点选“主节点”，从节点设置为“拾取”，Shift+鼠标左键依次点选从节点，选择需要耦合的自由度（U1、U2、U3对应UX、UY、UZ）；

⑤点击“确定”按钮，完成平动约束的定义。

关键字说明：

当Type=translation时:

失效绑定






失效绑定（TieBreak）的定义（对应上图）：

①主功能工具栏点击“连接”图标；

②点击“新建”连接；

③连接类型选择“失效绑定”；

④分别定义主/从面，定义方式可以选择“基于模型”的定义方式，此时需要确定单元为二维网格还是三维网格，Shift+鼠标左键选择目标面网格作为主/从面。也可以采用“基于已有面”的定义方式，点击“选择面”，在弹出窗口中选择之前定义的面即可。同时可以定义”失效力”，连接处的失效力超过设置的阈值时，连接失效；

⑤点击“确定”按钮，完成失效绑定的定义。

关键字说明：

当Type=TieBreak时：

第一行：1）从面，2）主面；

Mx文件示例：

*connector, type = tiebreak, FailureType = N2S_FailureOnly

Surf_1,connect_Set_4

1e+08,1e+08
