球铰






球铰（SphericalJoint）的定义（对应上图）：

①主功能工具栏点击“连接”图标；

②点击“新建”连接；

③连接类型选择“球铰”；

④依次定义节点1、节点2，其中节点1从属于PART-A，节点2从属于PART-B，在空间坐标上，节点1、2重合，这样方可实现PART-A和PART-B的球铰连接定义；

⑤点击“确定”按钮，完成球铰的定义。

关键字说明：

当Type=SphericalJoint时：

第一行：1）节点1，2）节点2，3）阻尼因子（默认值1.0），4）罚因子调整项（默认值1.0）；

Mx文件示例：

*connector, type = sphericaljoint

5000,5001,1,1

*connector, type = extranode

Part02,nset13

*connector, type = extranode

Part01,nset14

*set, type = node, name = nset13

5000

*set, type = node, name = nset14

5001

（蓝色部分为集合，详集合定义章节）
