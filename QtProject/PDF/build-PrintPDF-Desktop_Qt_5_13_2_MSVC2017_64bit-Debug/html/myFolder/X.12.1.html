点对面接触






点对面接触（NODETOSURFACE）的定义（对应上图）：

①主功能工具栏点击“接触”图标；

②点击“新建”接触；

③接触类型选择“点对面接触”；

④分别定义主/从面，点击“选择面”，在弹出窗口中选择之前定义的面即可。同时可以定义“摩擦系数”和“接触阻尼”，若考虑单元失效，则应该勾选“侵蚀接触”；

⑤点击“确定”按钮，完成点对面接触的定义。

关键字说明：

1.参数行

当type=NODETOSURFACE时：

第一行：

表面名称,块组成的表面名称。

2.子关键字

*friction:

为摩擦参数，是接触部分的子关键字，可以缺省；

a.参数行

第一行：

静摩擦系数，动摩擦系数。

*ContactDamping::

为接触阻尼定义，是接触部分的子关键字，可以缺省

a.参数行

第一行：

阻尼系数。

*twopass:

为双向接触定义，是接触部分的子关键字，可以缺省，当type=NODETOSURFACE，此关键字无效。

a.无参数行

Mx文件示例：

*contact, type = nodetosurface, name = Contact_1

Node1, surface_5

*friction

0.1, 0.1
