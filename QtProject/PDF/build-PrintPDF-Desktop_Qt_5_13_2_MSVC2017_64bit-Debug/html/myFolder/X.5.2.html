壳截面

壳截面定义步骤（对应上图）：

①主功能工具栏点击“截面”图标，进入截面属性定义及赋予面板，属性名称默认为“Property_1”；

②属性分类栏选择“壳截面”；

③材料列表中选择已经创建的“Material_1”；

④厚度设置为“0.28”，积分个数设置为“3”，点击“确定并赋予”完成截面属性的一键赋予。

关键字说明：

*shellsection:指明是壳体截面属性

第一行：

1.厚度

2.厚度方面的积分点个数

3.分层标记（0：没有材料角度；1：需要设置材料角度）

第二行：

1.若分层标记=1，每个积分点对应的材料角度（一行最多八个参数）

Mx文件示例：

*section, type = structural, name =Property_1, mat =Material_1

*shellsection

0.28, 3, 0
