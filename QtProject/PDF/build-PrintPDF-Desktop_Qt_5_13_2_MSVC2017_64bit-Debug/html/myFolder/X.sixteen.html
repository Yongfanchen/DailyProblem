面






面（Surface）的定义（对应上图）：

①主功能工具栏点击“面”图标；

②点击“新建”面；

③设置要创建的面类型，可选择“节点”或“块（segment）”面类型；

④对象来源方式选择“拾取”；

⑤作用对象选择“拾取”或者“通过面”；

⑥Shift+鼠标左键选择目标节点、二维单元、或三维单元表面，相应目标高亮显式，可以利用鼠标滚轮键对模型视图进行缩放，提高目标的选取精度。

⑦点击“确定”按钮，完成集合的定义。

关键字说明：

*Surface: 定义表面

1.必须定义的属性设置:

Type

当type=node时，定义由节点组成的表面。

当type=segment时，定义由块组成的表面。

Name

说明表面的名称。

2.可选的属性设置:

Object

当object=part时，表示该面的来源至部件。

当object=set时，表示该面的来源至单元集合。

当object=element时，表示该面来源至单元。

3.参数行

当object缺省时：

列举面的组成。

a.当type=node时，

第一行：

1、节点ID。

重复此行即可完成整个面的定义。

b.当type=segment时，

第一行：

1、块-节点1的ID，

2、块-节点2的ID，

3、块-节点3的ID，

4、块-节点4的ID，注意：当为三节点块时，置空。

4、块-节点4的ID，注意：当为三节点块时，置空。

当object=part时：

列举组成面的部件。

第一行：

1、部件名称。

重复此行即可完成整个面的定义。

当type=Spherical时

定义极轴上两点a、b。

第一行：

1、z轴点a的坐标。

2、z轴点b的坐标

当object=set时：

列举组成面的集合。

a.当type=node时，

第一行：

1、节点集合名称。

重复此行即可完成整个面的定义。

b.当type=segment时，

第一行：

1、单元集合名称。

重复此行即可完成整个面的定义。

当object=element时：

列举组成面的单元。

a.当type=node时，

第一行：

1、单元ID。

重复此行即可完成整个面的定义。

b.当type=segment时，

第一行：

1、单元ID，

2、面ID，

重复此行即可完成整个面的定义。
