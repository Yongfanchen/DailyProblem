传感器单元






传感器(Sensor)单元的定义步骤（对应上图）

①主功能工具栏点击“单元”图标；

②点击“传感器单元”进入传感器单元定义窗口；

③传感器类型设置为“时间传感器”；

④时间阈值设置为“4”ms（根据实际单位制设置），目的是在4ms时触发传感器；

⑤点击“确定”按钮，生成时间传感器单元

关键字说明：

当type=Sensor时，用于定义传感器单元（sensorPart）

第一行：1）传感器编号，2）传感器类型type，3）SBSFL

当type=1：第二行：1）节点ID；2）自由度；3）ACC；4）ATIME

当type=2：第二行：1）卷收器ID；2）PULRAT；3）PULTIM

当type=3：第二行：1）TIME

当type=4：第二行：1）节点1ID；2）节点2ID；3）DMX；4）DMN

Mx文件示例：

*element, type = sensor, part = sensorPart

18, 3, 0
