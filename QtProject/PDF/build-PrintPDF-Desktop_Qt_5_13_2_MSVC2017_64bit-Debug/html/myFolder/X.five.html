截面属性

三维实体截面






三维实体截面定义步骤（对应上图）：

①主功能工具栏点击“截面”图标，进入截面属性定义及赋予面板，

②属性名称修改认为“solid”(自定义)；

③属性截面栏选择“三维实体截面”；

④材料列表中选择已经创建的“Steel”(材料定义操作详材料定义章节)；

⑤点击“确定”完成截面属性的一键赋予；

⑥可以在左侧看到名称为“solid”的属性；

⑦点击左下角属性列表中的“赋予”按钮；

⑧弹出赋予属性窗口,点击“拾取”按钮；

⑨Shift+鼠标左键框选模型，可以看到实体单元高亮显式；

⑩点击“赋予”按钮，完成截面属性的赋予。

注：第5步也可以点击“确定并赋予”，将模型中所有的单元都赋予该属性，适用于模型中所有单元及材料皆相同的情况。

关键字说明：

*solidsection:指明是实体截面属性

无属性及参数

Mx文件示例：

*section, type=structural, name=solid, mat=Steel

壳截面

壳截面定义步骤（对应上图）：

①主功能工具栏点击“截面”图标，进入截面属性定义及赋予面板，属性名称默认为“Property_1”；

②属性分类栏选择“壳截面”；

③材料列表中选择已经创建的“Material_1”；

④厚度设置为“0.28”，积分个数设置为“3”，点击“确定并赋予”完成截面属性的一键赋予。

关键字说明：

*shellsection:指明是壳体截面属性

第一行：

1.厚度

2.厚度方面的积分点个数

3.分层标记（0：没有材料角度；1：需要设置材料角度）

第二行：

1.若分层标记=1，每个积分点对应的材料角度（一行最多八个参数）

Mx文件示例：

*section, type = structural, name =Property_1, mat =Material_1

*shellsection

0.28, 3, 0

梁截面

1.积分梁截面






积分梁截面定义步骤（对应上图）：

①主功能工具栏点击“截面”图标，进入截面属性定义及赋予面板，新建属性名称“intbeam_PID6”(自定义)；

②属性截面栏选择“梁截面”；

③分类栏选择“积分梁截面”；

④材料列表中选择已经创建的“mat3”(材料定义操作详材料定义章节)；

⑤点击“确定”完成截面属性的定义；

⑥可以在左侧看到名称为“intbeam_PID6”的属性；

⑦点击左下角属性列表中的“赋予”按钮，弹出赋予属性窗口,方式设置为“拾取”；

⑧Shift+鼠标左键点选模型，可以看到梁单元高亮显式

⑨点击“赋予”按钮，完成截面属性的赋予。

关键字说明：

当Type=Integrated时：

第一行:

1.积分规则：1表示一点积分；2表示2*2高斯积分；3表示3*3高斯积分

2.截面形状：0表示矩形；1.0表示圆型

第二行：

1.节点1的外径，或矩形s方向边长；

2.节点2的外径，或矩形t方向边长；

3.节点1的内径，或矩形s方向边长；

4.节点2的内径，或矩形t方向边长。

Mx文件示例：

*section, type = structural, name =intbeam_PID6, mat = mat3

*beamsection, type = Integrated

0, 1

1, 1, 0, 0

2.离散梁截面






离散梁截面定义步骤（对应上图）：

①主功能工具栏点击“截面”图标，进入截面属性定义及赋予面板，新建属性名称“Property_8”；

②属性截面栏选择“梁截面”；

③分类栏选择“离散梁截面”；

④材料列表中选择已经创建的“Discrete_Beam”(材料定义操作详材料定义章节)；

⑤点击“确定”完成截面属性的定义；

⑥可以在左侧看到名称为“Property_8”的属性；

⑦点击左下角属性列表中的“赋予”按钮，弹出赋予属性窗口,方式设置为“拾取”；

⑧Shift+鼠标左键点选模型，可以看到梁单元高亮显式

⑨点击“赋予”按钮，完成截面属性的赋予。

关键字说明：

当Type=Discrete时：

1.离散梁单元的局部坐标系更新准则:

为-3：基于节点1的角速度更新，计算剪切力导致的转矩

为-2：基于节点1的角速度更新，计算剪切力导致的转矩，将r轴调整为沿节点1指向节点2

为-1：基于节点1的角速度更新

为 0：基于节点 1与节点2 的平均值更新

为 1：基于节点2的角速度更新

为 2：基于节点2的角速度更新，计算剪切力导致的转矩，将r轴调整为沿节点1指向节点2

为 3：基于节点2的角速度更新，计算剪切力产生的转矩

2.离散梁单元的体积

3.质量惯性矩

Mx文件示例：

*section, type = structural, name =Property_8, mat = Discrete_Beam

*beamsection, type = Discrete

0, 1, 0

SPH截面






SPH截面定义步骤（对应上图）：

①主功能工具栏点击“截面”图标，进入截面属性定义及赋予面板，

②新建属性名称“sphsection3_PID3”；

③属性截面栏选择“SPH截面”；

④材料列表中选择已经创建的“mat3”(材料定义操作详材料定义章节)；

⑤定义SPH截面参数；

⑥点击“确定”完成截面属性的定义；

⑦点击左下角属性列表中的“赋予”按钮，弹出赋予属性窗口,方式设置为“拾取”；

⑧Shift+鼠标左键点选模型，可以看到SPH单元高亮显式

⑨点击“赋予”按钮，完成截面属性的赋予。

关键字说明：

*SPHsection:指明是SPH单元截面属性

参数行（两行）

第一行：

1.初始粒子间距：若为零，则自动计算，默认零。

2.初始光滑长度计算用系数（CSLH)：不得小于1，取1.2-1.5之间最合适，取值大于1.5后会带来很大的计算成本，默认1.2。

3.初始光滑长度（smoothLengthIni）：若为零，则通过粒子初始间距和CSLH计算，默认零。

4.最小光滑长度比例因子(hMinScale）：用于限制光滑长度的最小值，默认1.0。

5.最大光滑长度比例因子（hMaxScale）：用于限制光滑长度的最大值，默认1.0。

6.计算开始时间（startTime）：控制当前属性的部件计算开始时间，默认零。

7.计算终止时间（deathTime）：控制当前属性的部件计算终止时间，默认零。

第二行

1.XSPH系数：用于计算粒子的位移修正项，取值尽量小于1，默认零。

2.部件间粒子耦合标志：默认=1不参与耦合，前提是sphControl中的isPartsInac为1。

3.粒子所在Box的id：默认0则无Box限制。

Mx文件示例：

*section, type = structural, name =sphsection3_PID3, mat = mat3

*SphSection

0, 1.2, 0, 0.2, 2, 0, 1e+20

0, 1, 0
