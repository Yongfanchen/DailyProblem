刚体荷载






刚体载荷（rigidbody）的施加步骤（对应上图）：

①主功能工具栏点击“载荷”图标，点击“新建”载荷，名称为”Load_1”；

②载荷类型选择“刚体载荷”；

③定义刚体的y向竖向力和绕y轴的扭矩，若定义曲线，则x,y,z,rx,ry,rz为对应载荷的曲线调整系数；

④作用类型选择“拾取”，Shift+鼠标左键点选单元上的目标部件；

⑤可以看到部件外轮廓高亮显式，刚体载荷默认作用于刚体的质心；

⑥点击“确定”按钮，完成刚体载荷的施加。

关键字说明：

1.必须定义的属性设置

Type

当type=rigidbody时，用于施加刚体力或转矩，作用于单个刚体上。

Set

用于指定被施加边界条件的集合，需要与Type指定的载荷类型相匹配。

2.可选的属性设置

Name

用于说明本载荷的名称。

Fun

用于指定载荷的曲线。
