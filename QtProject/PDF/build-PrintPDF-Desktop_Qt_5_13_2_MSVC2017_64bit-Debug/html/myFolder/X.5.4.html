离散截面






离散截面定义步骤（对应上图）：

①主功能工具栏点击“截面”图标，进入截面属性定义及赋予面板，

②新建属性名称“discretesection1-PID3”；

③属性截面栏选择“离散截面”；

④材料列表中选择已经创建的“spring”(材料定义操作详材料定义章节)；

⑤点击“确定”完成截面属性的定义；

⑥可以在左侧看到名称为“discretesection1-PID3”的属性；

⑦点击左下角属性列表中的“赋予”按钮，弹出赋予属性窗口,方式设置为“拾取”；

⑧Shift+鼠标左键点选模型，可以看到离散单元高亮显式

⑨点击“赋予”按钮，完成截面属性的赋予。

关键字说明：

*discretesection:离散单元属性

无属性及参数。

Mx文件示例：

*section, type = structural, name =discretesection1_PID3, mat = spring

*discretesection
