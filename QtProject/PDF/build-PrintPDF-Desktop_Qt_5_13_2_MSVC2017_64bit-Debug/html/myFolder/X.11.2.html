节点刚体






节点刚体（MPC）的定义（对应上图）：

①主功能工具栏点击“连接”图标；

②点击“新建”连接；

③连接类型选择“节点刚体”；

④鼠标点击“主节点”，Shift+鼠标左键点选模型节点（1320）作为主节点；同理，鼠标点击“从节点”，Shift+鼠标左键点选模型节点（2641）作为从节点；

⑤点击“确定”按钮，完成节点刚体的定义。

关键字说明：

当Type=MPC时：

第一行：1）节点集合。

Mx文件示例：

*connector, type = mpc

connect_Set_7

*set, type = node, name = connect_Set_7
