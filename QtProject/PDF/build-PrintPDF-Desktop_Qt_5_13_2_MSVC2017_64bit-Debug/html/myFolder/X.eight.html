定义载荷

点荷载






点载荷（concentrated）的施加步骤（对应上图）：

①主功能工具栏点击“载荷”图标，点击“新建”载荷，名称为”Load_3”；

②载荷类型选择“点载荷”；

③定义点载荷的方向向量(0,-1,0表示沿y轴负向的方向向量)，若定义曲线，则x,y,z为对应载荷方向的曲线调整系数；

④点击“拾取”按钮；

⑤Shift+鼠标左键点选单元上的目标节点，可以看到节点高亮显式；

⑥点击“确定”按钮，完成点载荷的施加；

⑦可以看到点载荷的位置和方向。

关键字说明：

1.必须定义的属性设置

Type

当type=concentrated时，用于施加集中集中载荷，作用于节点集上。

Set

用于指定被施加边界条件的集合，需要与Type指定的载荷类型相匹配。

2.可选的属性设置

Name

用于说明本载荷的名称。

Fun

用于指定载荷的曲线。

Mx文件示例：

*load, type = concentrated, set = Loadrigidbody4,function = curve1, name = Load_3

1 ,0.0

2 ,-1.000000

3 ,0.0

*function, type = linear, name = curve1

0, 100

1, 100

面荷载






均布载荷（surfacedistribute）的施加步骤（对应上图）：

①主功能工具栏点击“载荷”图标，点击“新建”载荷，名称为”Load_3”；

②载荷类型选择“面载荷”；

③定义点载荷的方向向量(0,-1,0表示沿y轴负向的方向向量)；

④作用类型选择“三维网格表面”（对于二维网格，选择“二维网格表面”），点击“拾取”按钮；

⑤Shift+鼠标左键点选单元上的目标单元表面，可以看到单元表面高亮显式；

⑥点击“确定”按钮，完成面载荷的施加；

⑦可以看到点载荷的位置和方向。

关键字说明：

1.必须定义的属性设置

Type

当type=surfacedistribute时，用于施加均布载荷，作用于表面上。

Set

用于指定被施加边界条件的集合，需要与Type指定的载荷类型相匹配。

2.可选的属性设置

Name

用于说明本载荷的名称。

Fun

用于指定载荷的曲线。

Mx文件示例：

*load, type = surfacedistribute, set = Load_Set_8, name = Load_3

1,0,-1,0

重力






重力（gravity）的施加步骤（对应上图）：

①主功能工具栏点击“载荷”图标，点击“新建”载荷，名称为”Load_1”；

②载荷类型选择“重力”；

③定义重力的方向(0,-9.8,0表示重力荷载沿y轴负向)，若定义曲线，则x,y,z为对应载荷方向的曲线调整系数；

④作用类型选择“节点集合”，作用对象选择“通过零件”按钮；

⑤在弹出的零件窗口中选择待施加重力场的部件；

⑥点击“确定”按钮，完成重力的施加；

⑦可以看到重力的方向（带G的箭头）。

关键字说明：

1.必须定义的属性设置

Type

当type=gravity时，用与施加加速度载荷，作用于节点或单元集合上(隐式分析中，该项是重力加速度)。

Set

用于指定被施加边界条件的集合，需要与Type指定的载荷类型相匹配。

2.可选的属性设置

Name

用于说明本载荷的名称。

Fun

用于指定载荷的曲线。

Mx文件示例：(2表示Y方向)

*load, type = gravity, set = Load_Set_5, name = Load_1

2 ,-9.8

刚体荷载






刚体载荷（rigidbody）的施加步骤（对应上图）：

①主功能工具栏点击“载荷”图标，点击“新建”载荷，名称为”Load_1”；

②载荷类型选择“刚体载荷”；

③定义刚体的y向竖向力和绕y轴的扭矩，若定义曲线，则x,y,z,rx,ry,rz为对应载荷的曲线调整系数；

④作用类型选择“拾取”，Shift+鼠标左键点选单元上的目标部件；

⑤可以看到部件外轮廓高亮显式，刚体载荷默认作用于刚体的质心；

⑥点击“确定”按钮，完成刚体载荷的施加。

关键字说明：

1.必须定义的属性设置

Type

当type=rigidbody时，用于施加刚体力或转矩，作用于单个刚体上。

Set

用于指定被施加边界条件的集合，需要与Type指定的载荷类型相匹配。

2.可选的属性设置

Name

用于说明本载荷的名称。

Fun

用于指定载荷的曲线。

Mx文件示例：(2表示Y方向；5表示绕Y轴逆时针方向)

*load, type = rigidbody, set = Load_Set_7,function = , name = Load_1

2 ,1

5 ,10
