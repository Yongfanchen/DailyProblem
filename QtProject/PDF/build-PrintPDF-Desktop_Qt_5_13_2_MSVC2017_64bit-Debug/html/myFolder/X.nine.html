施加约束

位移/转角约束






位移/转角约束（displacement）的设置步骤（对应上图）：

①主功能工具栏点击“边界”图标，点击“新建”约束，名称为”BC_2”；

②边界条件类型选择“位移/转角”；

③为待选择的目标节点（或刚体）施加位移及转角约束，U1、U2、U3为系统坐标下的X，Y，Z方向的平动自由度；UR1、UR2、UR3为系统坐标下的绕X，Y，Z轴的转动自由度。0表示限制该自由度的平移或者转动，也可通过设置曲线对目标节点进行约束；

④对象类型选择“节点”或者“刚体”；

⑤作用对象选择“拾取”，Shift+鼠标左键点选或框选待约束区域节点；

⑥点击“确定”按钮，完成位移/转角约束的施加。

关键字说明：

1.必须定义的属性设置

Type

当type=displacement时，用于指定强制位移边界条件，仅限于显式分析类型.

Object

用于指定施加边界对象是刚体还是节点。

Set

用于指定被施加边界条件的集合。

2.可选的属性设置

Name

用于说明本边界条件的名称。

Function

用于指定施加边界条件的曲线。

Mx文件示例：

*boundary, type = displacement, object = node,set = nset3, name = BC_2

2,0

4,0

5,0

6,0
