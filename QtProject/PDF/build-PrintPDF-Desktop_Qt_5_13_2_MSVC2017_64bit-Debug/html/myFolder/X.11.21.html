壳-实体耦合






壳-实体耦合（ShellToSolid）的定义（对应上图）：

①主功能工具栏点击“连接”图标；

②点击“新建”连接；

③连接类型选择“壳-实体耦合”；

④Shift+鼠标左键点选与实体节点位置重叠（非共节点关系）的“壳节点”，实体节点设置为“拾取”，Shift+鼠标左键自上而下或从下到上按顺序选择实体节点；

⑤点击“确定”按钮，完成壳-实体耦合的定义。

关键字说明：

当Type=ShellToSolid时:

第一行：1）壳单元上参与绑定的节点，2）实体单元上参与绑定的节点集合。

Mx文件示例：

*connector, type = shelltosolid

2652,nset8

*set, type = node, name = nset8

1331, 1320, 1309, 1298, 1287

（蓝色部分为集合，详集合定义章节）
