万向节






万向节（UniversalJoint）的定义（对应上图）：

①主功能工具栏点击“连接”图标；

②点击“新建”连接；

③连接类型选择“万向节”；

④依次定义节点1~节点4，其中节点1和节点3从属于PART-A，节点2和节点4从属于PART-B。在空间坐标上，节点1、2重合，且由节点1、3组成的向量与有节点2、4组成的向量在空间上相互垂直。这样方可实现PART-A和PART-B的万向节连接定义；

⑤点击“确定”按钮，完成万向节的定义。

关键字说明：

当Type=UniversalJoint时:

第一行：1）节点1，2）节点2，3）节点3，4）节点4，5）阻尼因子，6）罚因子调整项；

重复此行完成多个万向节的定义。

Mx文件示例：

*connector, type = universaljoint

7100,7101,7102,7103,1,1

*set, type = node, name = nset16

7100,7102

*set, type = node, name = nset17

7101,7103

*connector, type = extranode

Part01,nset16

*connector, type = extranode

Part02,nset17

（蓝色部分为集合，详集合定义章节）
