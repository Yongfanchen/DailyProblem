额外节点






额外节点（extranode）的定义（对应上图）：

①主功能工具栏点击“连接”图标；

③连接类型选择“额外节点”；

④鼠标点击“刚体”，Shift+鼠标左键点选择刚体模型；点选“多节点”，Shift+鼠标左键点选刚体模型外部的节点（非刚性节点）；

⑤点击“确定”按钮，完成额外节点的定义。

注：在定义转动副、球铰、圆柱副，移动副等连接件时，需要首先定义额外节点。

关键字说明：

当Type=extranode时：

第一行：1）刚体，2）节点集合

Mx文件示例：

*connector, type = extranode

part3,nset7

*set, type = node, name = nset7

5000,5002
