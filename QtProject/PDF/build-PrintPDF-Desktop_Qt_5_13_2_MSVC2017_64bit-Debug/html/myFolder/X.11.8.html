移动副






移动副（TranslationalJoint）的定义（对应上图）：

①主功能工具栏点击“连接”图标；

②点击“新建”连接；

③连接类型选择“移动副”；

④依次定义节点1~节点6，其中节点1、节点3及节点5从属于PART-A，节点2、节点4及节点6从属于PART-B，节点1和节点3定义圆柱副的中心线，节点5、6定义中心线外的节点。在空间坐标上，节点1、2重合，节点3、4重合，节点5、6重合。这样方可实现PART-A和PART-B的移动副连接定义；

⑤点击“确定”按钮，完成圆柱副的定义。

关键字说明：

当Type=TranslationalJoint时:

第一行：1）节点1，2）节点2，3）节点3，4）节点4，5）节点5，6）节点6，7）阻尼因子（默认值1.0），8）罚因子调整项（默认值1.0）； 重复此行完成多个移动副的定义

Mx文件示例：

*connector, type = translationaljoint

7000,7002,7001,7003,7004,7005,1,1

*set, type = node, name = nset8

7000,7001,7004

*set, type = node, name = nset9

7002,7003,7005

*connector, type = extranode

Part1,nset8

*connector, type = extranode

Part2,nset9

（蓝色部分为集合，详集合定义章节）
