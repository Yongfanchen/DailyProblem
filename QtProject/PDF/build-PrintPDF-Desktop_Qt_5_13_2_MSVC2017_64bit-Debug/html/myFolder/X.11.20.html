节点绑定






节点绑定（TieNodes）的定义（对应上图）：

①主功能工具栏点击“连接”图标；

②点击“新建”连接；

③连接类型选择“节点绑定”；

④Shift+鼠标左键依次点选定义主节点及从节点集，定义“失效塑性应变值”，和“连接单元类型”，对于二维网格，应选择“壳单元”；

⑤点击“确定”按钮，完成节点绑定的定义。

关键字说明：

当Type=TieNodes时:

第一行：1）节点集合，2）失效时的节点等效塑性应变阈值（默认为0，若为默认值，则不考虑失效），3）连接单元类型：0代表壳单元，1代表实体单元。

Mx文件示例：

*connector, type = tienodes

constrained_tied_nodes16,1,0

*set, type = node, name = constrained_tied_nodes16

1261,2688,2689,2690

（蓝色部分为集合，详集合定义章节）
