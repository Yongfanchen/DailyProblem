初始条件

初始速度






初始速度（initial velocity）的设置步骤（对应上图）：

①主功能工具栏点击“初始条件”图标，点击“新建”初始条件，名称为”Init_1”；

②初始条件类型选择“速度/角速度”；

③为待选择的目标部件节点施加初始速度，V1、V2、V3为系统坐标下的X，Y，Z方向的初始速度分量；若要定义初始角速度，则需要勾选“旋转角速度”，设置角速度（弧度制）、旋转轴坐标及法向向量；

④对象类型选择“节点”或者“刚体”；

⑤作用对象选择“拾取”，Shift+鼠标左键点选或框选待约束区域节点；

⑥点击“确定”按钮，完成加速度/角加速度约束的施加。

关键字说明：

1.必须定义的属性设置

Type

当type=stress，用于给指定的单元集合设置初始应力值。

2.可选的属性设置

Set

用于指定设置初始条件的集合，可以为单元集合、节点集合，但是必须与type适配的集合一一对应。

3.参数行：

当type=velocity时，必须指定三个方向的速度值。

可以增加*rotate子关键字，用于追加定义旋转角速度

*rotate:

追加定义初始的旋转角速度

参数行

1）角速度，2）旋转轴x起始坐标，3)旋转轴y起始坐标，4）旋转轴z起始坐标，5）旋转轴法向量x分量，6）旋转轴法向量y分量，5）旋转轴法向量z分量

Mx文件示例：

*initial, type = velocity, set = Initial_Set_12

0, 10, 0

*rotate

5, 0,0,0, 0,1,0
