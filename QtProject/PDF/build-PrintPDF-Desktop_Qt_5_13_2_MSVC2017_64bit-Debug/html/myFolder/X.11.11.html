齿轮传动






齿轮传动（GearJoint）的定义（对应上图）：

①主功能工具栏点击“连接”图标；

②点击“新建”连接；

③连接类型选择“齿轮传动”；

④依次定义节点1~节点6，其中节点1、节点3及节点5从属于PART-A，节点2、节点4及节点6从属于PART-B，节点1、3和节点2、4分别定义主、从齿轮的法向，节点1、5和节点2、6分别定义主、从齿轮所在平面。这样方可实现PART-A和PART-B的齿轮传动连接定义；

⑤点击“确定”按钮，完成齿轮传动的定义。

关键字说明：

当Type=GearJoint时：

第一行：1）节点1，2）节点2，3）节点3，4）节点4，5）节点5，6）节点6，7）阻尼因子（默认值1.0），8）罚因子调整项（默认值1.0）；

第二行：1）两齿轮分度圆半径之比，2）主动轮分度圆半径。

直齿轮或斜齿轮传动时，参数1）为从动轮半径/主动轮半径；锥齿轮传动时，参数1）为主动轮半径/从动轮半径。

Mx文件示例：

*connector, type = GearJoint

5875, 5877, 5876, 5878, 5879, 5880, 1.000000,1.000000

1.000000, 0.000000

*set,type = node, name = nset1

5875, 5876, 5879

*set,type = node, name = nset2

5877, 5878, 5880

*connector, type = extranode

Part03, nset1

*connector, type = extranode

Part04, nset2
