后处理

1.云图显式设置


①求解完成，软件会自动跳转至后处理“云图”界面；“当前帧”默认是1.1，将当前帧设置为2.2，可以看到第二帧的位移云图，如图所示；

②“变量”默认是“Displacement”，即位移,“分量”默认为为“Mag”，即显示合位移；

③点击当前帧，将当前帧切换到显示“21.21”，即求解的最后一步；

④点击确定，即可显示最后一步的合位移云图；

⑤可通过修改“离散”度数值，即云图的颜色层次数，将数值修改为“20”；

⑥云图颜色离散度为“20”的显示，如图所示。


Mx文件示例：

*output, mode = mxdb, name = FieldOutput_1, timeInterval = 10

*field, object = element, variable = all

2.动态效果查看


①鼠标点击后处理菜单栏的“动画”图标，并点击左下角的“生成动画”按钮；

②点击“播放”按钮即可进行易拉罐跌落动态效果的的展示；

③修改“帧速率”窗口数据可实时对动画播放速度进行调整。

3.时程曲线查看






时程曲线的查看（对应上图）：

①后处理功能面板点击“XY图表”图标，进入时程曲线输出窗口；

②点击新建XY图表，名称为“CurveView_1”；

③变量选择“Stress（应力）”因变量选择“MISES（等效应力）”；

④Shift+鼠标左键在模型上指定时程曲线输出测点；

⑤可以看到测点高亮显示；

⑥点击“生成曲线图”；

⑦弹出时程曲线如上图所示，该曲线无属性，鼠标左键双击曲线空白区域进入曲线编辑窗口；


⑧勾选“格栅显示”，会在曲线上增加格栅；


⑨选择X轴，定义X轴标题为“时间”，点击“确定”按钮，完成横坐标轴名称的定义；


⑩选择Y轴，定义Y轴标题为“等效应力”，点击“确定”按钮，完成纵坐标轴名称的定义；


⑪最终生成的时程曲线如图所示。
