速度/角速度






速度/角速度（velocity）的设置步骤（对应上图）：

①主功能工具栏点击“边界”图标，点击“新建”约束，名称为”BC_1”；

②边界条件类型选择“速度/角速度”；

③为待选择的目标节点（或刚体）施加速度及角速度约束，V1、V2、V3为系统坐标下的X，Y，Z方向的速度分量；VR1、VR2、VR3为系统坐标下绕X，Y，Z轴的角速度分量（弧度制）。也可通过设置曲线对目标节点进行约束；

④对象类型选择“节点”或者“刚体”；

⑤作用对象选择“拾取”，Shift+鼠标左键点选或框选待约束区域节点；

⑥点击“确定”按钮，完成速度/角速度约束的施加。

关键字说明：

1.必须定义的属性设置

Type

当type=velocity时，用于指定强制速度边界条件，仅限于显式分析类型。

Object

用于指定施加边界对象是刚体还是节点。

Set

用于指定被施加边界条件的集合。

2.可选的属性设置

Name

用于说明本边界条件的名称。

Function

用于指定施加边界条件的曲线。

Mx文件示例：

*boundary, type = velocity, object = node,set = BC_Set_8, name = BC_1
