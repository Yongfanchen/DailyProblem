平动约束






平动约束（translation）的定义（对应上图）：

①主功能工具栏点击“连接”图标；

②点击“新建”连接；

③连接类型选择“平动约束”；

④Shift+鼠标左键点选“主节点”，从节点设置为“拾取”，Shift+鼠标左键依次点选从节点，选择需要耦合的自由度（U1、U2、U3对应UX、UY、UZ）；

⑤点击“确定”按钮，完成平动约束的定义。

关键字说明：

当Type=translation时:

第一行：1）平动约束作用节点集合，2）约束的自由度ID，ID=1，约束节点集合x方向平动自由度；ID=2，约束节点集合y方向平动自由度；ID=3，约束节点集合z方向平动自由度；ID=4，约束节点集合x和y方向平动自由度；ID=5，约束y和z方向的平动自由度；ID=6，约束x和z方向的平动自由度；ID=7，约束节点集合x、y和z方向的平动自由度。

Mx文件示例：

*connector, type = translation

NODESETCONSTN7, 7

*set, type = node, name = NODESETCONSTN7

1309, 1320, 1331, 2630, 2641, 2652

（蓝色部分为集合，详集合定义章节）
