膜单元










膜单元定义和关联步骤（对应上图）：

①主功能工具栏点击“单元”图标；

②点击“膜”进入单元选择窗口；

③点击“四边形”单元；

④点击选择“MBQ4”单元；

⑤Shift+鼠标左键框选膜单元，可以看到膜单元高亮显式；

⑥点击“确定”按钮，完成单元信息的关联。

关键字说明：

膜单元

当type=MBT3时，用于定义三角形膜单元，共三个节点

第一行：1）单元编号，2）节点编号1，3）节点编号2，4）节点编号3 重复此行完成膜单元部件定义。

当type=MBQ4时，用于定义全积分四边形膜单元，共四个节点

第一行：1）单元编号，2）节点编号1，3）节点编号2，4）节点编号3，5）节点编号4 重复此行完成膜单元部件定义。

当type=MBQ4R时，用于定义一点积分四边形膜单元，共四个节点

第一行：1）单元编号，2）节点编号1，3）节点编号2，4）节点编号3，5）节点编号4 重复此行完成膜单元部件定义。

当type=MITC4时，用于定义MITC4壳单元，共四个节点

第一行：1）单元编号，2）节点编号1，3）节点编号2，4）节点编号3，5）节点编号4 重复此行完成壳单元部件定义。

Mx文件示例：

*element, type = MBQ4R, part = sphereshell, section = shellsection1_PID1

1, 1, 10, 11, 2

2, 2, 11, 12, 3
