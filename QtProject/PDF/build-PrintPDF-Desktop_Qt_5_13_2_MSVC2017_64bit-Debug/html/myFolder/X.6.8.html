安全带单元






安全带(Seatbelt)单元定义和关联步骤（对应上图）

①主功能工具栏点击“单元”图标；

②点击“安全带单元”进入安全带单元定义窗口；

③按住Ctrl键，依次在模型上点击两个节点，可以看到节点高亮显式；

④在零件名中输入part3(自定义)；

⑤点击“确定”按钮，可以看到在两节点间生成安全带单元。

关键字说明：

当type=SeatBelt时，用于定义安全带单元

第一行：1）单元编号，2）节点1编号，3）节点2编号，3）卷收器编号，4）初始松弛长度 重复此行完成安全带单元部件的定义。

Mx文件示例：

*element, type = SEATBELT, part = part3, section = Property_2

15,2729,2631, 0, 0
