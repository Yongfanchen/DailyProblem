云图显式设置


①求解完成，软件会自动跳转至后处理“云图”界面；“当前帧”默认是1.1，将当前帧设置为2.2，可以看到第二帧的位移云图，如图所示；

②“变量”默认是“Displacement”，即位移,“分量”默认为为“Mag”，即显示合位移；

③点击当前帧，将当前帧切换到显示“21.21”，即求解的最后一步；

④点击确定，即可显示最后一步的合位移云图；

⑤可通过修改“离散”度数值，即云图的颜色层次数，将数值修改为“20”；

⑥云图颜色离散度为“20”的显示，如图所示。


Mx文件示例：

*output, mode = mxdb, name = FieldOutput_1, timeInterval = 10

*field, object = element, variable = all
