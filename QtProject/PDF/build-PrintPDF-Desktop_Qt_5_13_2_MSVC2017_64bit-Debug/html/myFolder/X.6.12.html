张紧器单元






张紧器(Pretensioner)单元的定义步骤（对应上图）

①主功能工具栏点击“单元”图标；

②点击“张紧器单元”进入张紧器单元定义窗口；

③选择张紧器类型，本算例为“力限制张紧器”；

④设置传感器编号（前面定义的）；

⑤关联卷收器编号，定义张紧器的加载力-拉出量曲线以及张紧器触发时间；

⑤点击“确定”按钮，完成张紧器单元的定义。

关键字说明：

当type=Pretensioner时，用于定义张紧器单元（pretensionPart）

第一行：1）张紧器编号；2）张紧器类型；3）传感器1编号；4）传感器2编号；5）传感器3编号；6）传感器4编号

当type=1：第二行：1）卷收器ID；2）TIME；

*load：定义卷收器加载曲线的子关键字

参数行

第一行：1）单元拉出量，2）力

重复此行完成力-拉出量的加载曲线定义

注意：该关键字必须定义

Mx文件示例：

*element, type = pretensioner, part = pretensionerPart

92, 1, 16, 0, 0, 0

17, 5

*load

0, 0

100, 10
