卷收器单元






卷收器(Retractor)单元的定义步骤（对应上图）

①主功能工具栏点击“单元”图标；

②点击“卷收器单元”进入卷收器单元定义窗口；

③传感器1设置为16（之前定义的时间传感器16，4ms时触发卷收器），2~4可不设置；

④输入与安全带单元相邻的梁单元节点编号（38），以及卷收器相邻的第一个安全带单元编号（15）。注意此梁单元节点编号不在安全带上，不可共节点，该节点主要起到卷收器对安全带的初始约束及定位作用；

⑤设置卷收器中安全带的加载力-位移曲线、卸载力-位移曲线，以及给进长度（卷收器中储存的安全带单元总长）；

⑤点击“确定”按钮，完成卷收器单元的定义。

关键字说明：

当type=Retractor时，用于定义安全带卷收器单元（retractorPart）

第一行：1）单元编号，2）卷收器节点编号，3）安全带单元编号，4）传感器1编号，5）传感器2编号，6）传感器3编号，7）传感器4编号

第二行：1）TDEL，2）PULL，3）LFED

*load：定义卷收器加载曲线的子关键字

Mx文件示例：

*element, type = retractor, part = retractorPart

93, 38, 15, 16, 0, 0, 0

0,0,20

*load

0, 0

10,10

*unload

0, 0

10,10
