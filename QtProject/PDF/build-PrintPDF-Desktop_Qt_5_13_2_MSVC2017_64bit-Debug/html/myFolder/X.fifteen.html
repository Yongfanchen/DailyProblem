局部坐标系






局部坐标系（CoordinateSystem）的定义（对应上图）：

①主功能工具栏点击“坐标系”图标；

②点击“新建”局部坐标系，名称为“CS_1”；

③设置局部坐标系“类型”为“直角坐标系”，设置方式为“三点”定义的方式

④依次用Shift+鼠标左键在模型上指定“原点”、“X轴上一点”及“XY平面上一点”；

⑤若选择的XY平面上一点关于原点与X轴正交，则XY平面上一点定义Y轴，确定X轴及Y轴后可由右手定则确定Z轴；

⑥点击“确定”按钮，完成局部直角坐标系的定义。

关键字说明：

*CoordinateSystem：定义局部坐标系

1.必须定义的属性：

Type

当type=Rectangular时，定义直角坐标系

当type=Cylindrical时，定义柱面坐标系

当type=Spherical时，定义球面坐标系

Name

定义局部坐标的名称

2.可选定义的属性：

property (仅当type=Rectangular时可用，且为必选项)

当property = node时，用于说明该局部坐标系通过节点来定义

当property = system时，用于说明该局部坐标系通过三点坐标来定义

当property = vector时，用于说明该局部坐标系通过两个向量来定义

Update (仅当property = node时可用，为可选项)

当Update=Yes时，说明该局部坐标系每时每刻都在更新

当Update=No时，说明该局部坐标系不随时间更新

Rotation (仅当property = vector时可用，为可选项)

Rotation=节点ID，定义该局部坐标系随该节点转动

3.参数行

当type=Rectangular时

a.当property = node时：

第一行：

1、节点1（原点）的ID。

2、节点2（x轴上一点）的ID。

3、节点3（x-y平面上一点）的ID。

b.当property = system时：

第一行：

1、节点1（原点）的坐标x1/y1/z1。

2、节点2（x轴上一点）的坐标x2/y2/z2。

3、节点3（x-y平面上一点）的坐标x3/y3/z3。

c.当property = vector时：（原点默认为0/0/0）

第一行：

1、节点1（x轴上一点）的坐标x2/y2/z2。

2、节点2（x-y平面上一点）的坐标x3/y3/z3。

当type=Cylindrical时

定义极轴上两点a、b。

第一行：

1、z轴点a的坐标。

2、z轴点b的坐标

当type=Spherical时

定义极轴上两点a、b。

第一行：

1、z轴点a的坐标。

2、z轴点b的坐标

Mx文件示例：

*coordinatesystem, type = rectangular, name = CS_1, property = system

210, 100, 100, 210, 100, 50, 170, 100, 100
