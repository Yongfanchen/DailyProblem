SPH截面






SPH截面定义步骤（对应上图）：

①主功能工具栏点击“截面”图标，进入截面属性定义及赋予面板，

②新建属性名称“sphsection3_PID3”；

③属性截面栏选择“SPH截面”；

④材料列表中选择已经创建的“mat3”(材料定义操作详材料定义章节)；

⑤定义SPH截面参数；

⑥点击“确定”完成截面属性的定义；

⑦点击左下角属性列表中的“赋予”按钮，弹出赋予属性窗口,方式设置为“拾取”；

⑧Shift+鼠标左键点选模型，可以看到SPH单元高亮显式

⑨点击“赋予”按钮，完成截面属性的赋予。

关键字说明：

*SPHsection:指明是SPH单元截面属性

参数行（两行）

第一行：

1.初始粒子间距：若为零，则自动计算，默认零。

2.初始光滑长度计算用系数（CSLH)：不得小于1，取1.2-1.5之间最合适，取值大于1.5后会带来很大的计算成本，默认1.2。

3.初始光滑长度（smoothLengthIni）：若为零，则通过粒子初始间距和CSLH计算，默认零。

4.最小光滑长度比例因子(hMinScale）：用于限制光滑长度的最小值，默认1.0。

5.最大光滑长度比例因子（hMaxScale）：用于限制光滑长度的最大值，默认1.0。

6.计算开始时间（startTime）：控制当前属性的部件计算开始时间，默认零。

7.计算终止时间（deathTime）：控制当前属性的部件计算终止时间，默认零。

第二行

1.XSPH系数：用于计算粒子的位移修正项，取值尽量小于1，默认零。

2.部件间粒子耦合标志：默认=1不参与耦合，前提是sphControl中的isPartsInac为1。

3.粒子所在Box的id：默认0则无Box限制。

Mx文件示例：

*section, type = structural, name =sphsection3_PID3, mat = mat3

*SphSection

0, 1.2, 0, 0.2, 2, 0, 1e+20

0, 1, 0
