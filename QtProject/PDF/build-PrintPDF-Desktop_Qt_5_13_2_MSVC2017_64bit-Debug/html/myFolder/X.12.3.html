全局搜索接触






全局搜索接触（automatic）的定义（对应上图）：

①主功能工具栏点击“接触”图标；

②点击“新建”接触；

③接触类型选择“全局搜索接触”；

④分别定义主/从面，点击“选择面”，在弹出窗口中选择之前定义的面即可。同时可以定义“摩擦系数”和“接触阻尼”，若考虑单元失效，则应该勾选“侵蚀接触”，对于双向接触类型，则应勾选“双向接触”，若考虑单个部件由于大变形产生的自接触，则应勾选“自接触”；

⑤点击“确定”按钮，完成全局搜索接触的定义。

关键字说明：

1.参数行

type=automatic时：

第一行：

缺省两个参数，但保留“,”

2.子关键字

*friction:

为摩擦参数，是接触部分的子关键字，可以缺省；

c.参数行

第一行：

静摩擦系数，动摩擦系数。

*ContactDamping::

为接触阻尼定义，是接触部分的子关键字，可以缺省

c.参数行

第一行：

阻尼系数。

*twopass:

为双向接触定义，是接触部分的子关键字，可以缺省，当type=NODETOSURFACE，此关键字无效。

c.无参数行

*contact, type = automatic, name = Contact_1, self = yes
