运动耦合






运动耦合（KinematicCoupling）的定义（对应上图）：

①主功能工具栏点击“连接”图标；

②点击“新建”连接；

③连接类型选择“运动耦合”；

④Shift+鼠标左键点选定义参考节点，然后点击“约束表面”，在弹出的对话框中选择定义的面，定义影响半径和待约束的自由度；

⑤点击“确定”按钮，完成对运动耦合的定义。

关键字说明：

当Type=KinematicCoupling时:

第一行：1）参考节点ID，2）约束影响半径（可缺省）。

第二行：1）被约束的节点表面的名称，2）被约束的第一自由度（若缺省，则约束全部自由度），

3）被约束的最后一个自由度（若缺省，仅约束参数指定的第一自由度）。

Mx文件示例：

*connector, type = kinematiccoupling

1287,10

surface_2,1,1

*surface, type = node, name = surface_2

2553

2564

2575

2586

2597
