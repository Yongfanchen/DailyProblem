面荷载






均布载荷（surfacedistribute）的施加步骤（对应上图）：

①主功能工具栏点击“载荷”图标，点击“新建”载荷，名称为”Load_3”；

②载荷类型选择“面载荷”；

③定义点载荷的方向向量(0,-1,0表示沿y轴负向的方向向量)；

④作用类型选择“三维网格表面”（对于二维网格，选择“二维网格表面”），点击“拾取”按钮；

⑤Shift+鼠标左键点选单元上的目标单元表面，可以看到单元表面高亮显式；

⑥点击“确定”按钮，完成面载荷的施加；

⑦可以看到点载荷的位置和方向。

关键字说明：

1.必须定义的属性设置

Type

当type=surfacedistribute时，用于施加均布载荷，作用于表面上。

Set

用于指定被施加边界条件的集合，需要与Type指定的载荷类型相匹配。

2.可选的属性设置

Name

用于说明本载荷的名称。

Fun

用于指定载荷的曲线。

Mx文件示例：

*load, type = surfacedistribute, set = Load_Set_8, name = Load_3

1,0,-1,0
