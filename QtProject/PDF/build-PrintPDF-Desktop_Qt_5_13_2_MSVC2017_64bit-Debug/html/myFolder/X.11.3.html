合并刚体






合并刚体（mergerigidbody）的定义（对应上图）：

①主功能工具栏点击“连接”图标；

②点击“新建”连接；

③连接类型选择“合并刚体”；

④鼠标点击“主刚体”，Shift+鼠标左键点选定义主刚体，主刚体外轮廓高亮显示；同理，鼠标点击“从刚体”，Shift+鼠标左键点选定义从刚体，从刚体外轮廓高亮显示；

⑤点击“确定”按钮，完成合并刚体的定义。

关键字说明：

当Type=mergerigidbody时：

第一行：1）主刚体，2）从刚体。

Mx文件示例：

*connector, type = mergerigidbody
