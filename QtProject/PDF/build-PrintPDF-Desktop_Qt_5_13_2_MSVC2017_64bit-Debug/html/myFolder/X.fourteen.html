曲线函数


曲线函数（Function）的定义（对应上图）：

①主功能工具栏点击“曲线”图标；

②点击“新建”曲线，曲线名称为“curve1”；

③曲线类型设置为“线性曲线”；

④重复点击“+”或就“-”可以增、减若干行，依次在曲线数据输入栏中输入X、Y散点数据，点击“显示曲线图”可以进行曲线预览

⑤点击“确定”按钮，完成曲线的定义。

关键字说明：

*function:定义函数信息

1.必须的属性设置：

Type

当type=constant，用于指定常数曲线。

当type=linear，用于指定分段线性曲线。

当type=smoothstep，用于光滑曲线。

Name

用于指定曲线名称

2.参数行：

当type=constant时：

第一行：1）常数值

仅一行完成曲线定义。

当type=linear时：

第一行：1）x值；2）y值

重复本行完成分段线性曲线定义

当type=smoothstep时：

第一行：1）x值；2）y值

重复本行完成光滑曲线定义

关键字对应关系

与INP：

MX::*function,type=linear对应INP::*AMPLITUDE,DEFINITION=TABULAR或INP::*AMPLITUDE

MX::*function,type=smoothstep对应INP::AMPLITUDE,DEFINITION= SMOOTH STEP

与K：

MX::*function,type=linear 对应K::*DEFINE_CURV

Mx文件示例：

*function, type = linear, name = curve1

0, 0

100, 0.001

200, 0.001
