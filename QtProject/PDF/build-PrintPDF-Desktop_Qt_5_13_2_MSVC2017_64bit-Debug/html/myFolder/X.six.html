单元类型的定义与关联

MX文件格式：*element, type =xxx, part =xxx, section = xxx

单元编号，单元节点1，单元节点2，单元节点3，…，单元节点n 后续重复完成部件的单元定义。

质量单元














质量（Mass）单元定义和关联步骤（对应上图）：

①主功能工具栏点击“网格”图标，在下面弹出的菜单栏中点击“创建网格”；

②网格形状选择“点”网格；

③部件名称设置为“M1”；

④Shift+鼠标左键点选单元上的任意节点，可以看到节点高亮显式；

⑤点击“确定”按钮，创建点单元

⑥主功能工具栏点击“单元”图标

⑦点击”质量”进入单元设置窗口，设置质量为0.01(同模型质量单位制)；

⑧Shift+鼠标左键点击之前创建的点网格的节点，可以看到节点高亮显式，点击“确定”按钮，在点网格上添加质量单元；

关键字说明：

质量单元

当type=mass时，用于定义质量单元：

第一行：1）单元编号，2）节点编号，3）附加的节点质量。 重复此行完成质量单元部件的定义。

Mx文件示例：

*element, type = MASS1, part = M1

2010, 1331, 0.01

滑轮单元






滑轮(Slipring)单元的定义步骤（对应上图）

①主功能工具栏点击“单元”图标；

②点击“滑轮单元”进入滑轮单元定义窗口；

③输入与安全带单元相邻的梁单元节点编号（126）。注意此梁单元节点编号不在安全带上，不可共节点，该节点主要起到滑轮对安全带的初始约束及定位作用；

④依次输入滑轮两侧的单元编号（上图为31、32）,滑轮和安全带的摩擦系数根据实际情况定义；

⑤点击“确定”按钮，可以看到在126节点处生成滑轮单元。

关键字说明：

当type=Slipring时，用于定义安全带滑轮单元（slipringPart）

第一行：1）单元编号，2）滑轮节点编号，3）安全带单元1编号，4）安全带单元2编号，4）摩擦系数

重复此行完成滑轮单元部件的定义

传感器单元






传感器(Sensor)单元的定义步骤（对应上图）

①主功能工具栏点击“单元”图标；

②点击“传感器单元”进入传感器单元定义窗口；

③传感器类型设置为“时间传感器”；

④时间阈值设置为“4”ms（根据实际单位制设置），目的是在4ms时触发传感器；

⑤点击“确定”按钮，生成时间传感器单元

当type=1：第二行：1）节点ID；2）自由度；3）ACC；4）ATIME

当type=2：第二行：1）卷收器ID；2）PULRAT；3）PULTIM

当type=3：第二行：1）TIME

当type=4：第二行：1）节点1ID；2）节点2ID；3）DMX；4）DMN

Mx文件示例：

*element, type = sensor, part = sensorPart

18, 3, 0

4

卷收器单元






卷收器(Retractor)单元的定义步骤（对应上图）

①主功能工具栏点击“单元”图标；

②点击“卷收器单元”进入卷收器单元定义窗口；

③传感器1设置为16（之前定义的时间传感器16，4ms时触发卷收器），2~4可不设置；

④输入与安全带单元相邻的梁单元节点编号（38），以及卷收器相邻的第一个安全带单元编号（15）。注意此梁单元节点编号不在安全带上，不可共节点，该节点主要起到卷收器对安全带的初始约束及定位作用；

⑤设置卷收器中安全带的加载力-位移曲线、卸载力-位移曲线，以及给进长度（卷收器中储存的安全带单元总长）；

⑤点击“确定”按钮，完成卷收器单元的定义。

关键字说明：

当type=Retractor时，用于定义安全带卷收器单元（retractorPart）

第一行：1）单元编号，2）卷收器节点编号，3）安全带单元编号，4）传感器1编号，5）传感器2编号，6）传感器3编号，7）传感器4编号

第二行：1）TDEL，2）PULL，3）LFED

*load：定义卷收器加载曲线的子关键字

Mx文件示例：

*element, type = retractor, part = retractorPart

93, 38, 15, 16, 0, 0, 0

0,0,20

*load

0, 0

10,10

*unload

0, 0

10,10

张紧器(Pretensioner)单元的定义步骤（对应上图）

当type=1：第二行：1）卷收器ID；2）TIME；

*load：定义卷收器加载曲线的子关键字

参数行

第一行：1）单元拉出量，2）力

重复此行完成力-拉出量的加载曲线定义

注意：该关键字必须定义

圆柱副






圆柱副（CylindricalJoint）的定义（对应上图）：

①主功能工具栏点击“连接”图标；

②点击“新建”连接；

③连接类型选择“圆柱副”；

④依次定义节点1~节点4，其中节点1和节点3从属于PART-A，节点2和节点4从属于PART-B，节点1和节点3定义圆柱副的中心线。在空间坐标上，节点1、2重合，节点3、4重合。这样方可实现PART-A和PART-B的圆柱副连接定义；

⑤点击“确定”按钮，完成圆柱副的定义。

关键字说明：

当Type=CylindricalJoint时:

第一行：1）节点1，2）节点2，3）节点3，4）节点4，5）阻尼因子（默认值1.0），6）罚因子调整项（默认值1.0）；

重复此行完成多个圆柱副的定义。

Mx文件示例：

*connector, type = cylindricaljoint

5910,5911,5912,5913,1,1

*connector, type = extranode

Part02,nset5

*connector, type = extranode

Part01,nset6

*set, type = node, name = nset5

5910,5912

*set, type = node, name = nset6

5911,5913

（蓝色部分为集合，详集合定义章节）
